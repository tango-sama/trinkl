# Beauty Store Landing Page (Arabic RTL)

A modern, responsive landing page for a beauty/cosmetics store, designed based on a specific visual mockup.

## Key Features
- **RTL Layout**: Fully optimized for Arabic language.
- **Responsive Design**: Works on Mobile, Tablet, and Desktop.
- **Custom Theme**: Using TailwindCSS custom colors to match the pink/mauve palette.
- **Components**:
  - Hero Section with circular image.
  - Features grid with icons.
  - Product Grid with cards.
  - Sticky Header and functional Footer.

## Maintenance
- When updating colors, modify the CSS variables in `index.html`.
- Product data is currently mocked in `utils/data.js`.
